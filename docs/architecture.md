# Архитектура AI n8n Assistant

## Идея
Входные данные → предобработка → запрос к LLM (OpenAI/YandexGPT) → пост-обработка → структурированный ответ.

## Узлы (черновой план)
- **HTTP Request** — интеграция с API
- **Function / Code** — предобработка/постобработка
- **IF / Switch** — ветвления логики
- **Set / Merge** — работа с полями и данными
- **Webhook / Trigger** — запуск воркфлоу
