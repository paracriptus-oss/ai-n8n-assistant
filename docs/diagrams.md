# Диаграммы

Ниже представлена схема логики работы типового AI-проекта с автоматизацией обработки данных.  
Она показывает этапы прохождения данных — от получения запроса до возврата готового результата, включая обработку ошибок.

```mermaid
flowchart LR
  A[Входные данные / Input] --> B[Предобработка / Preprocess]
  B --> C[Запрос к LLM / LLM Request]
  C -->|Успешно| D[Постобработка / Postprocess]
  C -->|Ошибка| F[Обработка ошибки / Error Handler]
  D --> E[Результат / Output]
  F --> G[Логирование / Повторный запуск]
